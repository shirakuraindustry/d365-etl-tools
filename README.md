# D365移行プロジェクト - 白倉工業

## 📘 プロジェクト名称
**D365移行プロジェクト**

## 🎯 プロジェクト目的
Microsoft Dynamics 365（D365）の導入を通じて、白倉工業の業務プロセスを標準化・効率化し、業務品質の向上と組織全体の高度化を図る。

D365の活用により、以下の改善を目指す：

- 生産性向上  
- データ利活用力の強化  
- 部門間連携の強化  
- 業務の属人化解消  
- データ品質向上  
- 長期的なデータ保存体制の確立  

## 🧭 プロジェクト背景
現状では部門ごとに個別システムを運用しており、情報の分散・業務自動化の阻害が発生。  
特に以下の課題が顕在化：

- 工事予定変更や外注工事に伴う事務処理の増加  
- 作業員スケジュール調整の複雑化  
- 担当者依存による属人化  
- 業務効率の低下とデータ整合性の欠如  

## 📦 プロジェクト範囲（スコープ）

対象部門：

- 工事営業部門  
- 工事発注・契約  
- 現場施工・記録  
- 請求・支払い  
- 安全関係書類提出業務（グリーンサイト連携）  
- 購買資材管理部門  
- 経理総務部門  

※D365のFinanceおよびSCMモジュールは本フェーズでは対象外

## 👥 ステークホルダーと体制

- **プロジェクトオーナー**：白倉工業 大江 知広  
- **プロジェクトマネージャー**：白倉工業 細内 雄太  
- **各部門代表者**：要件定義、受入テスト、業務移行を担当

## 📅 スケジュール（主要マイルストーン）

## ✅ 期待される便益

- 業務処理時間30%以上削減  
- 属人化の排除  
- データ品質向上（エラー・不整合抑止）  
- 法令対応強化（電子帳簿保存法等）  
- 意思決定の迅速化  
- 長期保存対応（10年以上）  
- コスト管理強化（予算超過アラート対応）
